# Space-Mission-Java
A console-based Space Mission Simulator built using Java.

## Features

- Rocket Name Input
- Fuel Level Input
- Destination Input
- Mission Details Dashboard
- Launch Validation System
- Menu-Driven Interface

## Concepts Used

- Java
- Scanner Class
- if-else Statements
- switch-case
- do-while Loop

## Future Updates

- Refuel System
- Mission History
- Countdown Timer
- Random Mission Events
